## Todo App
    
## Features:
* A simple react-typescript project of the todo app.
* Here, we can add a to-do task with the deadline.
* We can delete the task from the list when finished.

## Technology used:
* ReactJS
* Bootstrap
* TypeScript
* Netlify Hosting
